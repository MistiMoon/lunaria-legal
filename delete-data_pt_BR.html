Exclusão de dados do Lunaria

Última atualização: 9 de julho de 2026

O Lunaria não cria contas de usuário

O aplicativo não exige cadastro, login, senha ou e-mail. Dados do perfil, como nome, data e hora de nascimento, são armazenados localmente. Ao gerar uma leitura personalizada, alguns dados são enviados temporariamente aos serviços Firebase e Gemini. A última leitura e os dados necessários para limites podem ser armazenados no Firebase sob um identificador técnico do dispositivo.

Excluir dados do telefone

Abra seu perfil no aplicativo e escolha sair/excluir perfil. Você também pode limpar os dados nas configurações do Android ou desinstalar o Lunaria.

Solicitar exclusão de dados da nuvem

Envie um e-mail com o assunto “Exclusão de dados Lunaria”. Podemos solicitar informações adicionais para localizar os dados. A solicitação será atendida em até 30 dias.
